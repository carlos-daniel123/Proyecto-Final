## Proyecto Final de Python ##

Este es mi proyecto final de Python pro, es un bot de discord que responde comandos con tematica del calentamiento global

## ¿Que hace? ##

Este bot de discord entre varias cosas hace:

- te recomienda una dieta amigable con el medio ambiente
- te dice la huella de carbono que produces segun tu transporte
- y te dice el consumo de tu computadora aunque *puede ser que no funcione en todos los sistemas operativos*

## ¿Como instalarlo? ##

Instalarlo es muy simple, solo tienes que bajar el archivo ZIP, descomprimirlo y ejecutar el archivo .py dentro

## Dependencias ##

- Python (3.11+ recomendado)
- discord.py
- python-dotenv (archivo .env con token y un .env.example recomendado)
- un bot de discord y su token a la mano
  
Espero que disfrutes este bot de discord y le des un buen uso, este es el final de este README, muchas gracias por leer!!!
